Notes that swim the way you think

Features

Small creature, big capabilities

Everything you need to organize your world, nothing you don't.

🌊

Hierarchical notes

Organize notes in an infinite tree. Each note can have children, with configurable sort order — alphabetical or manual positioning.

🦐

Typed note schemas

Define custom note types with Rhai scripts. Fields support text, numbers, dates, booleans, selects, ratings, and more.

🐚

User scripts

Each workspace stores its own Rhai scripts. Create, edit, enable, and reorder scripts from the built-in script manager.

🪸

On-save hooks

Scripts can compute derived fields automatically — auto-generate titles, calculate durations, or set status badges on save.

🏷️

Tags

Add free-form tags to any note from the info panel. A tag cloud in the sidebar gives a workspace-wide overview, and the search bar matches tags alongside titles and text fields.

🔗

Note links

The note_link field type links one note to another. An inline search picker lets you find the target note; view mode renders a clickable navigation link. Query backlinks with get_notes_with_link().

🔍

Fuzzy search

Live search with debounced fuzzy matching across titles, text fields, and tags. Keyboard-navigable results that scroll into view.

📦

Export & Import

Export an entire workspace as a .zip archive, optionally password-protected with AES-256 encryption. Encrypted archives are automatically detected on import.

🔒

Encrypted at rest

All workspaces are AES-256 encrypted via SQLCipher. A password is required to create or open each workspace. Passwords can optionally be cached for the duration of a session. No account or cloud connection required — your data stays on your device.

💻

Cross-platform

Runs on macOS, Linux, and Windows via Tauri v2. Native performance backed by Rust and SQLite.

📎

File attachments

Attach any file to a note with drag-and-drop. Attachments are encrypted at rest alongside the workspace. Images show a thumbnail; all files can be downloaded on demand. Included in workspace export and import.

💬

Hover tooltips

Hovering a tree node for 600 ms shows a compact tooltip without navigating away. Mark any field with show_on_hover: true for instant inline previews, or define an on_hover hook to return fully custom HTML.

🌐

Internationalisation

Ships with 7 language packs: English, German, French, Spanish, Japanese, Korean, and Simplified Chinese. Switch languages live from Settings — dates and numbers automatically follow the locale's conventions.

🎨

Theming

Switch between Light, Dark, and System modes from Settings. Create and share custom .krilltheme files from the Manage Themes dialog — preview, edit, import, and apply themes without restarting.

📋

Operations log

Every mutation is recorded in an immutable log — browse history, filter by type or date, and purge old entries.

Extensible

Your notes, your types, your rules

Define custom note schemas with Rhai scripts. Add fields, hooks, and computed values — no recompilation needed.

  • Define schemas with typed fields (text, date, select, rating, note_link, file…)
  • Auto-derive titles and computed fields with on_save hooks
  • Custom view rendering with on_view hooks
  • Hover tooltips via on_hover hook or show_on_hover field flags
  • Embed attached images with display_image() or {{image: …}} markdown syntax
  • Query notes by type, tag, backlink, or attachment metadata
  • Ready-to-use templates — book collection, Zettelkasten, and more

task.rhai 
schema("Task", #{
    title_can_edit: false,
    fields: [
        #{ name: "name",   type: "text",   required: true  },
        #{ name: "status", type: "select",
           options: ["TODO", "WIP", "DONE"] },
        #{ name: "due_date", type: "date" },
    ],
    on_save: |note| {
        let status = note.fields["status"];
        let symbol = if status == "DONE" { "✓" }
            else if status == "WIP" { "→" }
            else { " " };
        note.title = "[" + symbol + "] " + note.fields["name"];
        note
    }
});
